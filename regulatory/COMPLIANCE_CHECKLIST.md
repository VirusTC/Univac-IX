# Hospital Readiness Checklist: Individual Compassionate Use Activation

This protocol outlines the operational and legal clearances required before a participating medical center may pull an experimental batch of Synthetic Fresh Whole Blood (SFWB) or Bio-Synthetic Marrow from inventory.

## 1. Pre-Administration Documentation Requirements
Before scheduling a patient infusion or graft under individual patient compassionate use, the attending physician must verify the presence of four core files in the electronic health record (EHR):
1. [ ] **Executed FDA Form 3926:** The single-patient expanded access application.
2. [ ] **Manufacturer Letter of Authorization (LOA):** The document generated via the Univac-IX system granting the physician permission to reference the core platform's active Investigational New Drug (IND) files.
3. [ ] **IRB Chairperson Concurrence Signature:** Concurrence signature by the Institutional Review Board Chair or a designated board member. Full board review is waived under Box 10b of Form 3926.
4. [ ] **Military Command Authorization Token:** (Required for active service personnel only under CCCRP rules).

## 2. Bedside Verification Protocol
Immediately prior to hanging the product bag on the infusion stand, the administering nurse must scan the container's passive RFID tag against the bedside terminal to ensure:
*   The batch identification number matches the exact patient assignment code inside the Univac ledger.
*   The batch status code displays strictly as `STATUS_ACTIVE_CCCRP_CLEARED`.
*   The hourly temperature telemetry log shows zero excursions above **6.0°C** during terminal transit.
